# E-Commerce Store (Jewellery Store)

## 📌 Overview
This project is a full-stack e-commerce web application designed for a jewellery store. It allows users to browse products, view product details, and interact with the store interface. The backend handles routing, data models, and server logic using Node.js and Express.

## 🚀 Features
- Product listing and display
- Product detail pages
- Organized backend structure using MVC pattern
- RESTful routes for handling requests
- Static frontend served from the server
- Scalable project structure

## 🛠️ Technologies Used
- Node.js
- Express.js
- JavaScript
- HTML, CSS
- MongoDB (if used)
- Mongoose (if used)

## 📂 Project Structure
- `models/` – Database models
- `routes/` – Application routes
- `public/` – Frontend files (HTML, CSS, JS)
- `server.js` – Main server file
